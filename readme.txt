1. 开发环境：windows
2. ide：Qt Creator(下载地址 http://download.qt.io/archive/qt/ 本项目使用qt5.8)
3. windows qt creator使用mysql请参考 http://lbbit.com/?p=19
4. GD-release.zip为该项目编译的release版本，在任何机器上可以直接双击打开
5. sql目录中为mysql数据库文件
6. 运行之前需要修改源码中的dbconnection.cpp文件中数据库相关配置，如需要release版本直接将编译好的release版本exe文件替换GD-release.zip压缩包中的exe文件
7. rtmp流媒体服务器搭建参考http://lbbit.com/?p=197
8. frp工具使用方法：
   环境搭建demo：
   https://blog.csdn.net/qq_25351621/article/details/78947477
   一个比较全面的工具使用介绍：
   https://studygolang.com/p/frp

* 本程序为门锁客户端软件，需要配合门锁端以及服务器端才可测试所有功能
